# Notes App 📝

## 🔹 Description
A simple and interactive Notes application to help users **create, edit, and delete notes**.  
Built to practice **JavaScript, DOM manipulation, and local storage**.

---

## 🔹 Features
- Add new notes with title and description
- Edit existing notes
- Delete notes
- Notes are saved in browser **local storage** (persistent across reloads)
- Responsive design for desktop and mobile

---

## 🔹 Tech Stack
- HTML
- CSS
- JavaScript
- Browser Local Storage
